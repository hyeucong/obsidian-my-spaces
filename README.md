**Managing what shows in your file tree shouldn't be hard and clunky.** 
*My Spaces* is a clean, minimal workspace manager that lets you filter your Obsidian sidebar to show exactly what you need, exactly when you need it.

## ✨ Features
- **File Tree Filtering:** Hide the clutter. View only the files and folders assigned to your active space.
- **Top Bar Navigation:** Instantly switch between spaces using custom icon buttons injected seamlessly into your file explorer header.
- **Auto-Tracking:** New notes or folders created inside an active space are automatically added to that space.
- **Customization:** Browse and set custom Lucide icons for every space.
- **Fast Management:** Right-click any file or folder to instantly add or remove it from a space.
- **Keyboard Friendly:** Assign custom hotkeys to jump straight into specific spaces without touching the mouse.

## 🚀 How to Use
1. Click the **"+"** button at the top of your file explorer to create a new Space.
2. Right-click any folder or file in your vault.
3. Select **"Manage spaces"** -> **"Add to [Your Space]"**.
4. Click your Space's icon at the top of the file explorer to filter your view!

## 📦 Installation

**From Obsidian Community Plugins (Recommended)**
1. Open Obsidian **Settings** -> **Community Plugins**.
2. Disable "Safe Mode" if prompted.
3. Click **Browse** and search for `My Spaces`.
4. Click **Install**, then **Enable**.

**Manual Installation**
1. Download the latest release from the [GitHub Releases](https://github.com/hyeucong/my-spaces/releases) page.
2. Extract the `main.js`, `manifest.json`, and `styles.css` files.
3. Place them in your vault's `.obsidian/plugins/my-spaces/` folder.
4. Reload Obsidian and enable the plugin in settings.
