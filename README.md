# My Spaces

My Spaces is a lightweight Obsidian plugin for organizing your workspace into focused spaces.

## Features
- Create and manage custom spaces for different projects or contexts
- Filter the file explorer to show only the files and folders in the active space
- Automatically include new files created inside a space
- Assign custom icons and quick navigation for faster switching

## Usage
1. Create a new space from the file explorer header.
2. Add folders or files to the space from the context menu.
3. Switch spaces to focus on the content you need.

## Installation
- Install from the Obsidian Community Plugins browser, or
- Download the latest release and place the plugin files in your vault's .obsidian/plugins/my-spaces folder.
