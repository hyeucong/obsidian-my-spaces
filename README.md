# My Spaces

Create clean, minimal spaces to control which files and folders appear in your sidebar. Managing your file tree shouldn't be hard or clunky.

## Features

- **Custom Spaces**: Separate your folder & file view with spaces.
- **Header Icon Bar**: Toggle spaces directly from the File Explorer header.
- **Command Palette**: Switch spaces instantly via hotkeys.
- **Context Menu**: Right-click any file or folder to assign it to a space.
- **Icon Suggestions**: Built-in icon picker with live previews in settings.

## Usage

1. **Create**: Click `+` in the header or add a space in settings.
2. **Assign**: Right-click any file or folder & add to space.
3. **Switch**: Click a space icon or run a command to filter your view.

## Installation
- **Community Plugins**: Search for **My Spaces** & **Install** then **Enable**.
- **Manual**: Place release files into `.obsidian/plugins/`.
